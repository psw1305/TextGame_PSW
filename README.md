# Shelter

생존자가 쉘터를 찾아가는 텍스트 게임입니다.

## ⚙️ 개발환경
* Visual Studio 2022
* .Net 7.0
* C# 11.0

## ⏰ 개발일정
* 2023.11.06 ~ 2023.11.13
